# AI-SDLC 1.0.0

AI-SDLC 1.0.0 提供一套本地优先、可恢复、可验证的 AI 原生研发工作流。

## 交付能力

- 新项目初始化与已有项目只读接入；
- Codex 项目规则和 PowerShell 偏好持久化；
- checkpoint、状态查看、安全预演与恢复；
- requirement、design-contract、implementation、frontend-evidence 四类工程闭环；
- 规则、约束、分支、任务和关闭条件门禁；
- 本地对抗 PR 审查、修复计划、复审与 CI 证明；
- 页面契约、浏览器探针、视觉回归和可访问性证据；
- AgentOps outbox、配置诊断和失败重试；
- Windows、macOS、Linux 源码与离线交付。

## 安装

```powershell
python -m pip install "git+https://github.com/SinclairPan/Ai_AutoSDLC.git@main"
ai-sdlc --version
```

## 初始化

```powershell
ai-sdlc init . --agent-target codex --shell powershell
ai-sdlc adapter status
ai-sdlc run --dry-run
```

## 离线包

- `ai-sdlc-offline-1.0.0-windows-amd64.zip`
- `ai-sdlc-offline-1.0.0-macos-arm64.tar.gz`
- `ai-sdlc-offline-1.0.0-linux-amd64.tar.gz`

## 验证

```powershell
uv run pytest -q
uv run ruff check src tests scripts
uv run ai-sdlc verify constraints
uv run python scripts/validate_public_release_identity.py .
uv build
```
